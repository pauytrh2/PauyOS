<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/pauytrh2/PauyOS.git
```

<p>2. Download toolchains</p>

```
scons toolchains
```

<p>3. Run</p>

```
scons run
```
